This is Hardware for Version 1.0 of my Extruder modification for printing silicone or other liquid materials using ender 3 base. 

This designe is created for standard 5ml leuer type syringe. 

this design uses linear guide rail and standard M4 threaded rod as a lead screw. 

It is not very precise because of some slack with in the place where syringe and piston are mounted.
I plan to improve it in the next version designing whole new system of syringe mounting allowing for mounting different sizes of syringes. 

I have had some good succesfull prints with this design, but keep in mind that this is still development/ prototype version.

IMPORTANT NOTICE! Keep in mind that depending on lenght of the needle that you use the end of the needle can be around 100mm LOWER IN Z AXIS than standard FDM printing nozzle. to archieve succesfull prints you need to move up the Z homing endstop higher or pres the endstop manualy before every print. aslo remember to adjust total print height in the software so the printer shall not exceed its physical limit and fall out the rails. 
Recomended Slicer is my own specially developed slicer Driplab SLicer which you will be able to find on my github. It is made exactly for this type of aplications and you have there plenty of tools helping you to simulate and have succesfull prints with this modification.
Other useful tools like syringe force calculator or cold extrusion g code generator that would be helpful when calibrating this desing you can also find on my github.
